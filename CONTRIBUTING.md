# Contributing

Thanks for helping build the collection! This repo is auto-generated from a
single source of truth, so the workflow is intentionally narrow.

## Status

Seedance 2.5 is in preview; the public launch is **early July 2026**. Until the
model is broadly available, entries are author-built **starter templates** that
demonstrate prompt structure. Once you can run the public model, please attach
a real output and credit the prompt's author.

## The one rule

**Only edit `prompts.json`.** The `README.md` and `README.zh-CN.md` files are
generated by `scripts/build_readmes.py` and will be overwritten on merge.

## Adding a prompt

1. Append a new entry to `prompts.json` under `entries`:

   ```json
   {
     "index": 9,
     "id": "9-short-kebab-slug",
     "title": { "en": "Short title", "zh": "简短标题" },
     "category": "text-to-video",
     "author": { "displayName": "Name", "xHandle": "handle_without_at" },
     "sourceUrl": "https://x.com/handle/status/...",
     "promptType": "template",
     "description": { "en": "One sentence.", "zh": "一句话。" },
     "prompt": "Full prompt text, verbatim.",
     "parameters": { "duration": "30s", "aspectRatio": "16:9", "references": 0 },
     "featured": false,
     "publishedAt": "2026-07-01",
     "toseaUrl": "https://tosea.ai/prompts/seedance-2-5#9-short-kebab-slug"
   }
   ```

2. Run the generator locally:

   ```bash
   python scripts/build_readmes.py
   ```

3. Commit the JSON edit and the regenerated READMEs in the same PR.

## Field notes

- **`category`** — one of: `text-to-video`, `image-to-video`,
  `multi-reference`, `region-edit`, `camera-cinematic`. Pick the closest fit.
- **`promptType`** — `template` for an untested scaffold, `verified` once you
  have generated a real output with the public model.
- **`author.xHandle`** — omit (use `null`) if you cannot verify. We do not
  fabricate handles.
- **`sourceUrl`** — prefer the author's own post. Use `null` for original
  templates.
- **`prompt`** — verbatim. Do not paraphrase, translate, or reformat.
- **`featured`** — reserved for editors. Set `false` in PRs.

## Translations

We ship English + Simplified Chinese. Provide both `en` and `zh` for `title`
and `description`. Keep the `prompt` itself in its original language.

## Code of conduct

Be kind, credit authors, and do not submit prompts you did not create or
verifiably discover from a public post. Seedance and Doubao are trademarks of
ByteDance; this is an independent community resource.
